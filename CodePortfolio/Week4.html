Parameters

R markdown documents can include one or more parameters whose values can be set for rendered reports. The following example is the sea ice data that I used for another project. I have set the parameters as; month = August, pole = North pole, and the year = 1979 with customized color fill.

library(rnoaa)
library(rgdal)
library(ggplot2)
urls <- seaiceeurls(mo = 'Aug', pole = 'N', yr = 1979)
out <- seaice(urls)
ggplot(out, aes(long, lat, group = group)) + 
  geom_polygon(fill = "skyblue1") + 
  theme_ice()
